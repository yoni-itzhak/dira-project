# בחירת דירה – סירקין 201

אפליקציה סטטית בעברית להשוואה ובחירת דירה בפרויקט אדרת בצמרת / סירקין 201.

## מה יש באפליקציה

- 105 דירות עם כל הפרמטרים שחולצו מתכניות ה־DWFx
- סינון לפי מחיר מטרה, בניין, חדרים, קומה ומסקנת הניתוח
- מיון לפי דירוג, נוף, תכנון, מחיר, שטח ורווח משוער
- כרטיס ועמוד פרטים לכל דירה עם חיתוך התכנית, יתרונות, חסרונות ובדיקות חובה
- סימון מועדפים הנשמר בדפדפן
- השוואה של עד ארבע דירות זו לצד זו
- רשימת הדירות המובילות ומפת סביבת הפרויקט
- הורדת קובץ ה־Excel המלא

## הפעלה מקומית

אין צורך ב־build או בהתקנת חבילות:

```bash
python3 -m http.server 8080
```

לאחר מכן פותחים `http://localhost:8080`.

## פרסום ב־GitHub Pages

ב־Settings → Pages בוחרים:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

## הערה חשובה

הציונים, המחירים והערכת הרווח הם כלי עזר לקבלת החלטה ואינם תחליף לתכנית מכר, מפרט, שמאות או ייעוץ משפטי. יש לאמת במיוחד מידות, מרפסות, חניה, מחסן, הצמדות, כיווני אוויר, מבנים עתידיים והצמדה למדד תשומות הבנייה.
